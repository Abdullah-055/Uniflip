UniFlip: Campus Exchange Hub
----------------------------------

Project Description:
UniFlip is a simple and user-friendly campus marketplace where students can buy, sell, and exchange goods and services.

How to Run:
1. Open the project folder.
2. Locate the "index.html" file.
3. Double-click to open it in your web browser (Chrome, Firefox, Edge, etc.).

For Developers:
1. Open the project folder in VS Code.
2. Install the "Live Server" extension (if not installed).
3. Right-click "index.html" and select "Open with Live Server" to run the project.

Enjoy using UniFlip!
